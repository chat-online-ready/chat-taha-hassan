دليل سريع — تشغيل التطبيق المحلياً
----------------------------------

1) فك الضغط (إن لزم) ثم افتح الطرفية داخل المجلد chat-online-ready
2) ثبّت الاعتمادات:
   npm install
3) ابدأ الخادم:
   npm start
4) افتح المتصفح وادخل:
   http://localhost:4000

ملاحظات للنشر أونلاين:
- يمكنك نشر المجلد كله إلى خدمة مثل Render, Railway, أو Heroku.
- هذه الخدمة تستخدم منفذ PORT من البيئة (process.env.PORT) لذلك تعمل على معظم منصات الاستضافة.
